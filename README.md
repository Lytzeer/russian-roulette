# Russian Roulette

## Introduction

This is the russian roulette game made in python

## Installation

### On Windows :

- Clone this repo :
```shell
git clone https://github.com/Lytzeer/russian-roulette.git
```

- Install requirements :
```shell
pip install -r requirements.txt
```

- Run the game :
```shell
python main.py
```

### On Linux :

- Clone this repo :
```shell
git clone https://github.com/Lytzeer/russian-roulette.git
```

- Install requirements :
```shell
pip3 install -r requirements.txt
```

- Run the game :
```shell
python3 main.py
```
